Selecting Items with jQuery

You can use the $ to select items from the DOM. It will feel a lot like selecting with querySelectorAll().

For example:

  • To select all tags:
    • $('a')
  • To select all elements from the class 'container':
    • $('.container')
  • To select all elements with the id 'special':
    • $('#special')

Also edit CSS properties with ease:

  • // start by grabbing the element you want to change
    var x = $('h1')

    // next call the css method.
    // the first parameter is the element you want to change,
    //the second is what you want to change it to.
    x.css('color', 'red')
